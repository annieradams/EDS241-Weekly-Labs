Weekly Lab Materials for EDS 241



## Description 
This repo contains the weekly lab materials for EDS 241, Environmental Policy Evaluation. Each folder contains the necessary data and coding template that students will need for our in lab coding each week. 

## Downloading materials 
You can download the github folder for each week by clicking on the weekly lab material link on the [course website]([https://eds-223-geospatial.github.io/](https://garberadamc.github.io/W26-Policy-Eval/)). This link utilizes the [DownGit](https://github.com/MinhasKamal/DownGit) tool to immediately download all necessary materials for each week in a zipper folder, rather than having to clone or fork the repository.
